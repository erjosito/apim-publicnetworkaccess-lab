then attempt to RE-ENABLE public access:
  CMD: az rest --method PATCH .../apim-iac-ven9pg?api-version=2024-05-01 \
         --body '{ "properties": { "publicNetworkAccess": "Enabled" } }'
  OUT ERROR (RequestDisallowedByPolicy):
    Resource 'apim-iac-ven9pg' was disallowed by policy 'apim-deny-pna-assign'.
    evaluatedExpressions: publicNetworkAccess Enabled NotEquals Disabled => True => Deny.

RESULT: SUCCESS. Deny blocks re-enabling public access (prevents drift). Deny CANNOT create
the disabled state itself - it only blocks non-compliant writes; pair it with IaC or DINE.
